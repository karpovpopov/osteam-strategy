01МИС → CRM — авто-импорт пациентов
Python-скрипт делает SQL-запрос к МИС ежедневно. Выписанные пациенты автоматически попадают в CRM — никакого ручного труда. Данные доступны сразу для запуска постоперационной цепочки.
02Чат-бот — приём лидов 24/7
ИИ — единый «мозг» для ВКонтакте, MAX и Telegram. Отвечает на вопросы об ОМС и другие FAQ. Горячие лиды мгновенно передаются администратору с уведомлением.
03SMM-агент — контент на автопилоте
Генерация постов с помощью ИИ. Автопубликация в ВКонтакте, Instagram, Telegram, MAX, Одноклассники, YouTube по расписанию на месяц вперёд. Участие персонала: 1–2 часа в неделю.
04Постоперационное сопровождение → сарафанное радио
Автоматическая цепочка сообщений: советы по реабилитации, запрос отзыва на Яндекс Картах и 2ГИС, реферальное предложение для родственников. Запускается автоматически при выписке из МИС.
05МИС-робот — автозаполнение историй болезни
Врач вводит краткую запись → ИИ структурирует данные → программный агент автоматически заполняет форму истории болезни в МИС. Реализуется в рамках отдельного проекта Фазы 4. Позволяет существенно сократить время на оформление документации.
Система запускается поэтапно. Каждая фаза добавляет новый уровень автоматизации и не требует остановки предыдущих блоков.
